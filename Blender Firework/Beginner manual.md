#  Blender Fireworks (Beginner Friendly)

A simple and beginner-friendly fireworks simulation made in Blender.
This project is designed to help you understand the basics of particles, animation, and emission.


##  What You Will Learn

* How to create a basic particle system
* How to simulate upward motion (like a rocket)
* How to create an explosion effect
* How to use emission for glowing particles

---

##  How It Works (Simple Explanation)

This fireworks effect is made in 2 main steps:

###  Launch (Going Up)

* A particle is emitted upward
* Velocity controls how fast it goes up

###  Explosion

* At a certain time, particles spread outward
* This creates the firework burst effect

---

## Try Changing These Values!

You can experiment by changing these:

* **Count** → Number of particles
   More = bigger explosion

* **Velocity** → Speed of particles
   Higher = faster movement

* **Lifetime** → How long particles exist
   Longer = bigger spread

* **Emission Strength** → Brightness
   Higher = more glow 

---

##  Beginner Experiments

Try these for fun:

*  Change colors randomly
*  Increase particle count for bigger fireworks
*  Add multiple explosions
*  Move the camera for cinematic effect

# Blenderで花火を作る（初心者向け）
Blenderで作成した、初心者にも分かりやすいシンプルな花火シミュレーションです。このプロジェクトは、パーティクル、アニメーション、発光の基本を理解するのに役立つように設計されています。

# 学習内容
基本的なパーティクルシステムの作成方法
ロケットのような上昇運動をシミュレートする方法
爆発効果の作り方
発光粒子にエミッションを使用する方法
仕組み（簡単な説明）
この花火効果は主に2つのステップで作成されます。

# 打ち上げ（上昇中）
粒子が上向きに放出される
速度は上昇の速さを制御する

# 爆発
ある時点で粒子が外側に拡散する
これにより花火の爆発効果が生まれます
これらの値を変更してみてください！
以下の項目を変更して試してみてください。

カウント→ 粒子の数　多いほど爆発は大きくなる

速度→ 粒子の速さ　数値が大きいほど動きが速い

寿命→ 粒子が存在する期間　長いほど拡散範囲が広がる

発光強度→ 明るさが高いほど、より明るく光ります

# 初心者向け実験
ぜひ試してみてください：

色をランダムに変更
より大きな花火を作るには、粒子数を増やす必要があります。
複数の爆発を追加する
映画のような効果を出すためにカメラを動かす
