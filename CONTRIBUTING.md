# 🤝 Contributing Guide

Thanks for contributing to Blender Fireworks!

---

## 📋 Workflow

1. Create an Issue
2. Assign a Priority (Triage → Critical → High → Medium → Low → Backlog)
3. Work on the task
4. Submit a Pull Request

---

## 🐛 Reporting Bugs

- Describe the issue clearly
- Include steps to reproduce
- Provide screenshots if possible

---

## ✨ Feature Requests

- Explain the idea
- Describe the benefit
- Keep it simple and clear

---

## 🔧 Pull Requests

- Keep changes focused
- Write clear commit messages
- Link related Issues

---

## 🎯 Rules

- Be respectful
- Keep things organized
- Follow the project structure
